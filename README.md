# Node.js Random Emoji Generator

Generate random emojis on the command line

## Random Emoji

```bash
$ node index.js
❤️
$ node index.js
🍎
```

## Emoji by name

```bash
$ node index.js strawberry
🍓
```

Invalid emoji names will print an error

```bash
$ node index.js strawberryyyy
Invalid emoji name
```
